# NormanPublicSchools
##Overview
This project focuses on building and improving district-wide library reporting systems across 24 K–12 schools. The goal was to enhance data accuracy, standardize reporting processes, and generate actionable insights for administrators, librarians, and district stakeholders.
🎯 Objectives
Standardize library data reporting across multiple schools
Improve survey structure and data quality
Build dashboards for administrative decision-making
Identify trends in library usage and engagement
🛠️ Tools & Technologies
Looker Studio
Tableau
Microsoft Excel
Data Cleaning & Validation Techniques
📈 Key Contributions
Built and deployed district-wide dashboards using Looker Studio, Tableau, and Excel, standardizing library reporting across 24 K–12 schools and supporting 30–50 stakeholders in administrative decision-making.
Improved data quality by refining survey structure and analyzing 100+ responses, increasing survey completeness by 5% and identifying inconsistencies in library usage and reporting trends.
Cleaned, validated, and structured survey datasets using Excel and Tableau to improve accuracy and reliability of district-wide reporting systems.
Analyzed dashboard and survey data to identify usage patterns and generate insights that informed district-level reporting and decision-making.
📊 Impact
24 schools supported through standardized reporting systems
100+ survey responses analyzed
5% improvement in survey completeness
30–50 district stakeholders using dashboards for decision-making
